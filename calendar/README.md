# Ats's Hindu Calendar

An interactive, responsive, and beautifully designed web-based Hindu Calendar widget and Full Month Viewer.

## Features
- **Widget View**: Displays the current date, Tithi, Nakshatra, Yoga, Karana, Rahu Kaal, Ritu, corresponding Vikram Samvat, and celestial signs.
- **Daily Panchang**: Quick dropdown access to today's detailed Panchang and upcoming events/festivals.
- **Full Calendar**: A grid-based month view depicting all days with indicators for Vrats, Festivals, Marriage, Naming, Travel, and Sankranti muhurats.
- **PWA Ready**: Can be installed as a Progressive Web App on mobile home screens.
- **Multi-lingual**: Supports English and Hindi translations (`EN`/`हिं`).
- **Theming Options**: Built-in color palettes to customize the visual gradient theme.

## Architecture
- Single `index.html` file encompassing structure, styling, and logic.
- Built with Vanilla JavaScript, HTML5, and **Tailwind CSS** (via CDN).
- Customized fonts (Marcellus, Poppins, Noto Sans Devanagari) via Google Fonts.

## Usage
Simply open `index.html` in your web browser. No compilation or server is required for basic functionality.
