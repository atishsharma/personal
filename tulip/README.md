# Tulip File Explorer

A beautiful, sleek, and high-performance cross-platform file manager download page, showcasing "Tulip File Explorer - Beautiful & Universal".

## Features Outline
- **Cross-Platform**: Provides direct download links for Windows (zip), Linux (deb), and macOS (arm64 zip).
- **Responsive Hero Section**: Eye-catching gradients and CSS blobs.
- **Features Highlight**: Outlines primary value propositions (Lightning Fast, Adaptive UI, Open & Secure).
- **Image Gallery**: Built-in simple lightbox gallery for displaying application screenshots.
- **Modern Design**: Designed using HTML/CSS built on Inter and Playfair Display fonts, featuring smooth glassmorphism effects.

## Project Structure
- `index.html`: The main landing page code for the application downloading site.
- `icon.png`: Main logo for the Tulip File Explorer.
- `image-*.jpg`: Assorted gallery images containing screenshots/showcases of the explorer.

## Usage
Simply open `index.html` in an internet browser.
