# EmberLoom

Welcome to **EmberLoom**—a modern Indian artisan brand reimagining heritage through the blend of slow-poured fragrances and meticulous hand-hooked crochet art. 

This repository serves as a collection of interconnected web applications and pages that power the EmberLoom digital presence, ranging from a premium storefront and a business intelligence dashboard to an interactive web-based design tool.

## 📂 Project Structure

The project is composed of three main web modules:

### 1. The Premium Storefront (`index.html`)
The primary landing page that introduces the "Soul of Karigari." 
- **Features:** 
  - Elegant, modern UI with scroll-reveal animations and glassmorphism elements.
  - Brand philosophy section highlighting the synergy of Kannauj botanicals and hand-hooked sleeves.
  - Interactive "Lookbook" gallery showcasing design studies.
  - "The Studio" storefront for artisanal products (e.g., The Ivory Karigari, Kesar Monogram).
  - Premium typography (Playfair Display & Plus Jakarta Sans).

### 2. Market Analysis Dashboard (`market.html`)
A dedicated business intelligence and fiscal strategy page for FY 2026. 
- **Features:** 
  - Dynamic KPI scoreboard for MSRP, Margins, and Target Markets.
  - Interactive data visualizations powered by **Chart.js**, including seasonal order projections, market pricing bubbles, and cost breakdown donut charts.
  - Comprehensive SWOT matrix (Strategic Matrix).
  - Social-First Sales Funnel analysis outlining the Instagram-to-WhatsApp checkout experience.

### 3. Ember Crochet Studio (`studio/index.html`)
An advanced, browser-based digital editor for creating custom crochet patterns and designs. 
- **Features:** 
  - **Grid Editor:** Customizable interactive grid (20x20 to 64x64) for drawing pixel-perfect crochet sequences.
  - **Silhouette Boutique:** Pre-defined templates spanning Fashion, Bags, Home Decor, Baby, and Amigurumi categories.
  - **Artisan Motif Library:** Over 30 ready-to-use stamp motifs, plus custom Text, Emoji, and Image stamping capabilities.
  - **Layer Management:** Full layer control (add, hide, move, delete) and an Undo/Redo history stack.
  - **Studio Preview:** A live 3D perspective mask viewer with tilt, zoom, and background color customization.
  - **Export:** Seamless one-click generation of PDF blueprints and design documents using **html2pdf.js**.

## 🛠️ Technologies Used
- HTML5 / CSS3
- **Tailwind CSS** (via CDN for rapid styling and responsive design)
- Vanilla JavaScript
- **Chart.js** (for market intelligence visual charts)
- **html2pdf.js** (for exporting studio designs)
- Google Fonts & FontAwesome

## 🚀 Getting Started

No build tools or Node.js environment are required to run this project. It is built entirely with client-side technologies.

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd ember
   ```
3. Open any of the HTML pages directly in your preferred web browser:
   - Double-click `index.html` for the landing page.
   - Double-click `market.html` for the business report.
   - Double-click `studio/index.html` for the crochet design studio.

Alternatively, use a local server like Live Server (VS Code extension) or Python's HTTP server for the best experience across local assets.
