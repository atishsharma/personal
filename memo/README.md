# Letterhead Editor (Memo)

A client-side web application for quickly generating and formatting printable letterheads and memos.

## Features
- **Live Preview Panel**: Edit your letterhead on the left and see the real-time A4 printable rendering on the right.
- **Adjustable Logo**: Sliders to control the logo size and the dimensions of the logo bounding box.
- **Rich Text Editing**: WYSIWYG buttons (Bold, Italic, Underline) for the Address/Header and the Body Content.
- **Print to PDF**: Built-in functionality to directly trigger the browser's print dialog, optimized for A4 paper size.

## Project Files
- `index.html`: The main web application interface.
- `logo.png`: Sample logo used in the header.

## Usage
1. Open `index.html` in your web browser.
2. Edit the address, reference number, and date directly in the preview or via the control panel input.
3. Write your content in the main body.
4. Click **Print PDF** to save or print the memo.
