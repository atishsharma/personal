Everything You Need

Pure Playback

Instantly scan and organize thousands of files. Full support for MP3, FLAC, WAV, and OGG formats with lossless quality.

YouTube Engine

Stream audio directly from YouTube with intelligent webm/opus caching for royalty-free, lightning-fast media handling.

Synced Lyrics

Sing along with perfectly timed, real-time synced lyrics powered by the LRCLib API integration.

Rich Integrations

Seamlessly scrobble your listening history to Last.fm and fetch rich metadata through our Spotify API integration.

Cross-Platform

A consistent, beautiful desktop environment built on Electron 30. Fully supports Linux and Windows natively.

Ultra Fast UI

Component-based UI built with React 18, Vite 5, and Tailwind CSS guarantees zero lag and a butter-smooth experience.
