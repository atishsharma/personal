CORE Teaching Protocol
  1. Slide open the programming cover.
  2. Press ENABLE (En). LCD displays En.
  3. Select Location: Press a Page digit then Key digit.
  4. Press DO (L). LCD shows L- (Learning mode).
  5. Send Signal: Use the IR Simulator. LCD shows C.
  6. Press OK (↵) to bond. LCD flashes J.
Advanced Programming
Combine multiple commands into a single keypress.

  1. Press ENABLE.
  2. Select a location (e.g., 0 then 1).
  3. Press DO (L).
  4. Enter sequence locations (e.g. P 8 a).
  5. Press OK (↵) to save.
Run commands automatically at a specific time.

  1. Press ENABLE.
  2. Press AT (Absolute) or WAIT.
  3. Set time using digit/rocker keys.
  4. Press DO (L).
  5. Enter location to execute.
  6. Press OK (↵) to set timer.
Review and modify existing programs.

  1. Press EDIT.
  2. Enter the location to review.
  3. Use ◀ / ▶ to move cursor.
  4. Press DELETE to erase character.
  5. Press OK (↵) to save changes.
