# My 3D Cordyceps Lab

An interactive 3D simulation and walkthrough of a Cordyceps mushroom growing laboratory, built using **Three.js**.

## Features
- **3D Environment Walkthrough**: First-person controls using `WASD` / Mouse to navigate the simulated laboratory environment.
- **Mobile Joystick Support**: Nipple.js integration provides a virtual joystick for mobile users.
- **Lighting Controls**: Switch between Day and Night modes, and toggle specific grow light spectrums (Purple, Warm, Cool).
- **Environment Interaction**: Hide specific objects (like walls or blackout cloths) using long-press or menu toggles.
- **Procedural Textures**: Features canvas-generated textures for brick walls and checkerboard floors for a lightweight payload. 

## Technology
- HTML5 / CSS3 / Vanilla Javascript
- [Three.js](https://threejs.org/) (r128)
- `PointerLockControls` & `OrbitControls`
- `nipplejs` for virtual joystick

## How to Run
Just open `index.html` in your browser. Wait for the initial loading text to disappear, then jump into the walkthrough mode using the on-screen menu.
